# CLAUDE.md

鉄棒着地チャレンジ — 大車輪から離手して着地を決めるブラウザゲーム。

## 構成

- 単一ファイル構成。`index.html` にHTML・CSS・JavaScriptをすべて含む。ビルド・依存関係・フレームワークなし。
- GitHub Pagesで公開中: https://makenico.github.io/tetsubo-game-iphone/
- 対象環境はiPhone Safari。タッチ操作（touchstart即反応、`touch-action:manipulation`、`user-select:none`）を壊さないこと。PCはスペースキー・クリックでも操作可。
- localStorageの使用は可（GitHub Pages公開済みのため解禁）。ベストスコア等の記録永続化に使う。

## 運用ルール

- 変更後は必ず `git commit` + `git push` すること（pushしないとGitHub Pagesに反映されない）。

## コードの構造（index.html内）

- ゲームモード: `idle` → `swing`（大車輪）→ `air`（滞空）→ `done`（着地）
- 重力設定 `GRAV`: 地球・月・木星の3モード。D得点係数が変わる
- 採点: D得点（宙返り数×重力係数+ボーナス+ひねり×0.5）+ E得点（着地10点満点）− 天井修理費。マット外は-4.0、踏ん張り忘れは-2.0
- ひねり: 空中で左右スワイプ（PCは←→キー）。最大5回。宙返り数×ひねり数の組み合わせで技名認定（`wazaName()`）
- タッチ入力の使い分け: 滞空中の高空はキャンバスタッチ=スワイプ判定、着地直前（今や!表示中）はタップ即踏ん張り。踏ん張りボタンは常時有効
- 審判コメントは `land()` 内でE得点・宙返り数・場外などに応じて分岐。同ティア3連続で特殊セリフ（`streakMsg`）に差し替え
- 効果音はWeb Audio APIで自前生成（外部ファイルなし）。`ensureAudio()` をユーザー操作内で呼びAudioContextをresume（iOS制約）
- 満点（E9.5以上・マット内）で紙吹雪＋観客スタンディング＋ファンファーレ

## 審判コメントの人格（厳守）

- E審判（辛口）のコメントは**関西弁**の人格を守ること。標準語に直さない。
- 辛口だがユーモアがあり、ツッコミ・比喩・擬人化（マット・床・重力がしゃべる等）を多用するスタイル。
- コメントを追加・変更する場合も既存のトーン（例:「マットの気持ち考えたことあるか!?」「重力『俺の負けや』」）に合わせる。
